# Flashcard App

A command-line flashcard quiz built in Python.

## How to run
python main.py

## What I learned
- How to identify when a function is doing too much
- Refactoring existing code without breaking it
- Writing unit tests with unittest
- Mocking input() with unittest.mock.patch
- Using side_effect to simulate user input in sequence
- Using GitHub Issues to track bugs
- Set up GitHub Actions to automatically run unit tests on every push 

## Features
- Type your answer directly — auto-checked instantly
- Hint system — reveals letters one at a time
- Shuffle cards before starting or practicing
- Practice mode for cards you got wrong
- Keeps asking until you get everything right
